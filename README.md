# JDBC CRUD Operations

This project demonstrates basic CRUD (Create, Read, Update, Delete) operations using Java JDBC with MySQL.

## Technologies Used
- Java
- JDBC
- MySQL

## Files
- RetrieveEmployee.java – Fetch employee records
- UpdateEmployee.java – Update employee details
- DeleteEmployee.java – Delete employee records

## Database Details
- Database: db583
- Table: Employee

This project is for learning and academic purposes.
